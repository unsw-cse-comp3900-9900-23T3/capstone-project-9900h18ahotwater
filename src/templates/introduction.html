Multi-label Classification

Automatic image tagging primarily assigns relevant tags to digital images. These tags are used to highlight prominent features and entities within the images. By uploading a set of pictures and text descriptions, we can observe the labels generated by each model. After uploading the files, you can click on 'Performance' to view the effectiveness of each model.

What did the backend model do?

Data Pre-processing

In our cool image-labeling tool, we break sentences into words, clean out the extra stuff, and simplify words. We then match words with unique codes or popular methods like 'Word2Vec'. For pictures, we make sure they're all the same size, adjust their brightness, and sometimes flip or crop them for variety. We ensure each image matches its description, split our data for training and testing, and mix features from different sources for better results.

Model - Ⅰ

The "Beyond Shared Subspace" model efficiently integrates multi-view data, including video screenshots, cover images, and descriptive text. In three pivotal steps, it first extracts essential features from each data type. Next, it merges these distinct features, enhancing their combined representation power. Finally, using individual and integrated features, it performs classification, producing a comprehensive output that leverages both visual and textual cues.
