VIT(vision transformer).

VIT is a network used to extract image features. Its original architecture diagram is shown below.The input image is first divided into many patches, in the paper An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale it's 16. Input the patch into the Embedding layer of Linear Projection of Flattened Patches, and get vectors, usually called 15 tokens. Then add a new token in front of a series of tokens. After that input it into the Transformer Encoder, which is shown as below, and stack the block L times. Transformer Encoder has as many outputs as inputs. Finally, the features of the image have been extracted.

BERT(Bidirectional Encoder Representations from Transformers)

BERT is a pre-trained natural language processing model that acquires rich language knowledge by learning a large amount of text data and can be used for various natural language processing tasks. In the BERT model, we can use its pre-trained model to extract text features. The input of the BERT model is a text sequence, which is usually processed by word segmentation and encoding. The output of the BERT model includes the word vector (Word Embeddings) of each word and the vector (Sentence Embeddings) for the entire sentence. We can use these word vectors and sentence vectors to build our own text feature vectors. Specifically, the BERT model can extract two types of text features: 1. Word Embeddings: The BERT model generates a corresponding word vector for each word in the input sequence. This word vector reflects the semantic information and contextual information of the word. 2. Sentence Embeddings: The BERT model generates a sentence vector in the entire sentence of the input sequence. This sentence vector reflects the semantic information and contextual information of the entire sentence.

Classification Network.

This part of the network is the last step before outputting results. Its function is to classify the features obtained from the sample in the above series of operations and output the final result. This is also the downstream work of the entire network, which can transform unreadable, incomprehensible features into categories. Directly fuse the features above (step 2) for classification (green). The text features extracted in step 1 are classified (yellow). The image features extracted in the first step are classified (blue). After the classification, directly average the three matrices, and the subsequent result (matrix) is the final classification result.
